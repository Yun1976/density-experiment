# 信息密度估计器 — 数学模型与迭代框架

> 基于香农信息论 + KL 散度 + 后悔驱动校准

---

## 一、形式化定义

### 1.1 信息密度（Information Density）

对于任务上下文中的一个信息块 $x$，其信息密度定义为：

$$\rho_t(x) = S(x) \cdot \lambda(t) \cdot R(x, \mathcal{H}_{\text{active}}) \cdot C(x, \mathcal{C})$$

其中：
- $S(x) = -\log_2 P(x)$ — **香农惊喜度**（Shannon surprise），$P(x)$ 是 $x$ 在当前任务假设树下出现的先验概率
- $\lambda(t) = \max(0.3, 1 - 0.2\cdot(t - t_{\text{first\_seen}}))$ — **S衰减因子**（v1.2，惊喜是一次性的）
- $R(x, \mathcal{H}_{\text{active}}) = \max_{h \in \mathcal{H}_{\text{active}}} \text{sim}(x, h)$ — **活跃假设相关性**（v1.1修正，仅取当前轮次活跃假设）
- $C(x, \mathcal{C})$ — **因果新颖性**，x 是否为因果图引入了新信息

**直觉**：一个信息"密度高" = 它既意外（低先验概率）、又相关（与核心假设紧密）、且新（带来新因果知识），但意外是一次性的——已知信息不再惊喜。

### 1.2 丢留判定（Drop/Keep Decision）

保留当且仅当：

$$\rho(x) > \theta_t$$

其中 $\theta_t$ 是第 $t$ 轮的密度阈值，随循环演化。

### 1.3 阈值迭代（Threshold Calibration）

$$\theta_{t+1} = \theta_t \cdot (1 - \alpha \cdot r_t + \beta \cdot d_t)$$

其中：
- $r_t \in [0,1]$ — 第 $t$ 轮的**后悔率**：上轮丢弃的信息中，本轮证明相关的比例
- $d_t \in [0,1]$ — 第 $t$ 轮的**冗余率**：上轮保留的信息中，累计 N 轮未引用的比例
- $\alpha, \beta$ — 学习率（初始建议：$\alpha=0.15, \beta=0.10$）

### 1.4 收敛性

系统存在不动点 $\theta^*$ 使得 $r_t \to 0$ 且 $d_t \to 0$。

证明思路：
- 若 $\theta$ 过高 → 丢弃过多 → 后悔率上升 → $\theta$ 下降
- 若 $\theta$ 过低 → 保留过多 → 冗余率上升 → $\theta$ 上升
- 两个方向都有恢复力 → 必收敛

## 二、可操作的密度评估流程

### 对每个信息块执行：

```
1. 计算 S(x):
   - 若 x 是预期内的工具输出: P(x)≈0.9 → S(x)≈0.15
   - 若 x 是常见推理路径: P(x)≈0.5 → S(x)=1
   - 若 x 是与假设冲突的反例: P(x)≈0.05 → S(x)≈4.3
   - 若 x 是命题级颠覆: P(x)≈0.001 → S(x)≈10

2. 计算 R(x, H):
   - 若 x 直接涉及假设树中某假设: R≈0.9-1.0
   - 若 x 相关但间接: R≈0.3-0.5
   - 若 x 无关: R≈0

3. ρ(x) = S(x) · R(x, H)

4. 判定:
   - ρ(x) > θ_t → 保留（写入 TASK.md）
   - ρ(x) ≤ θ_t → 压缩或丢弃
```

### 先验概率估计表（实际使用）

| 信息类型 | P(x) 估计 | S(x) |
|---------|----------|------|
| 工具调用成功（预期行为） | 0.95 | 0.07 |
| 标准推理路径结论 | 0.70 | 0.51 |
| 新发现的API行为 | 0.20 | 2.32 |
| 与预期不一致的结果 | 0.08 | 3.64 |
| 推翻了已验证假设 | 0.02 | 5.64 |
| 动摇了任务前提 | 0.001 | 9.97 |

## 三、校准记录模板

每轮压缩后记录：

```
## 密度校准日志 — 第 T 轮

### 本轮参数
- θ_T = [当前阈值]
- ρ_min(retained) = [保留的最低密度]
- ρ_max(dropped) = [丢弃的最高密度]

### 保留清单
| 信息块 | S(x) | R(x,H) | ρ(x) | 理由 |
|--------|------|--------|------|------|

### 丢弃清单
| 信息块 | S(x) | R(x,H) | ρ(x) | 丢弃理由 |

### 后悔日志（回顾第 T-1 轮丢弃）
| 被丢弃的信息 | 丢弃时 ρ | 本轮发现 | 后悔？ |
|-------------|---------|---------|--------|

### 冗余日志（回顾保留超过 3 轮的信息）
| 保留的信息 | 保留时 ρ | 累计未被引用轮数 | 冗余？ |
|-----------|---------|----------------|--------|

### 阈值校准
- 后悔率 r_T = [0-1]
- 冗余率 d_T = [0-1]  
- θ_{T+1} = θ_T · (1 - 0.15·r_T + 0.10·d_T) = [新阈值]
```

## 四、初始参数

| 参数 | 初始值 | 说明 |
|------|--------|------|
| $\theta_0$ | 1.0 | 初始阈值，偏保守 |
| $\alpha$ | 0.15 | 后悔学习率 |
| $\beta$ | 0.10 | 冗余学习率 |
| 冗余判定轮数 N | 3 | 保留后 N 轮未引用即视为冗余 |
| $\theta_{min}$ | 0.2 | 不可跌破的最低阈值 |
| $\theta_{max}$ | 5.0 | 不可突破的最高阈值 |
